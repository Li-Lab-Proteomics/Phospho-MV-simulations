# Phospho-MV-simulations
Evaluation of differential abundance analysis strategies for phosphoproteomics data with missing values

## Requirements
### R (4.0+)
* invgamma
* Bioconductor
* limma
* SDAMS
* pcaMethods
* trust
* ggplot2
* gridExtra

### Python (3.7+)
* NumPy (1.19.2+)
* Pandas (1.2.1+)
* Scikit-Learn (0.23.2+)
* SciPy (1.5.2+)
